# AI Product Photo Generator

Upload a product photo, choose a scene, and preview AI-generated e-commerce product images. Demo MVP for Amazon, Etsy, and Shopify sellers.

## What is this?

A demo landing page for an AI-powered product photo generation tool. Users can upload a product image, select from 6 scene templates, and see mock AI-generated results to explore what's possible before the full version launches.

## Features (v0.1 Demo)

- **Upload & Preview** — Drag and drop or click to upload product photos (browser-local, no server storage)
- **6 Scene Templates** — Clean White Background, Lifestyle on Table, In Hand, Nature/Outdoor, Luxury Minimalist, Holiday/Seasonal
- **Mock AI Generation** — 3-second simulated loading with progress bar
- **Result Gallery** — Preview 4 mock generated product photos
- **Pricing Preview** — Free demo / Pro plan / Business plan (waitlist only, no payment)
- **FAQ** — 8 common questions about the tool
- **Fully Responsive** — Works on desktop, tablet, and mobile

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the demo.

## Build

```bash
npm run lint
npm run build
```

## Important Notes

This is a **demo MVP** (v0.1). It does not include:
- Real AI image generation APIs
- remove.bg or any background removal service
- User authentication
- Database storage
- Payment processing (Stripe)
- Real image downloads (results are mock previews)

The full AI generation feature is in development. Join the waitlist on the pricing page for early access.
